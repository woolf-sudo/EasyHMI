# EasyHMI
基于专用SoC更容易实现人机交互。流畅的处理、炫酷的界面、丰富的外设资源，让人机交互变得更容易更炫酷。

---

# 1. 介绍

[EasyHMI](https://github.com/woolf-sudo/EasyHMI) 是基于HMI专用SoC硬件，搭配上嵌入式openrtos系统，通过C编程，实现的高性能人机交互demo；
非常适合对人机界面有需求的产品，例如： 智能家电、智能家居、车用仪表盘、工业人机交互等等；
相比通用串口屏，高性能MPU核心板，基于Android的界面，该实现方式更简单，显示效果更优化；
提供给用户丰富的API，功能丰富的UI控件，便捷的代码配置器，让使用者很容易二次开发，实现自己的产品。

## 1.1 主要特性

- 图形处理能力强（轻松支持 1080P）；
- 丰富的LCD接口（TTL/MIPI/LVDS/CPU/SPI）；
- 周边扩展性高（MAC/USB/CAN/SPI/UART/SDIO/SAR ADC/）
